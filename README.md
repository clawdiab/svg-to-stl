# svg-to-stl

Convert SVG files to STL 3D models by extruding 2D paths into 3D solids.

## Features

- Extrude SVG paths into 3D STL files
- Configurable extrusion depth
- Supports complex paths, curves, and shapes
- CLI and Python API

## Install

```bash
pip install -r requirements.txt
```

## Usage

```bash
python svg_to_stl.py input.svg --depth 5 --output output.stl
```
